Repository plugin: My Course Files, Moodle 2.x 

This module pretends to facilitate access to courses files associated with a user  quickly, so you can reuse the files of their courses and prevent re-upload it in the activities.

This Moodle 2.x repository plugin should be installed into the 'repository' directory within your
Moodle installation.
